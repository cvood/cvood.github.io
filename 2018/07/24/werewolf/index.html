狼人杀 大致思路

七月 24, 2018

大致流程

玩家

玩家扮演三种角色，普通村民、狼人、特殊职业（神）。
村民只能投票，狼人可以在夜间杀人，神可以再夜晚发动各种特殊能力。

village

  • 属性
    • id
    • role
    • alive
    • good
    • police
  • 方法
    • vote

werewolf

  • 属性
    • vilage.属性
  • 方法
    • vote
    • werewolf vote

seer

  • 属性
    • village。属性
  • 方法
    • vote
    • 查验身份

开始游戏

游戏房间

  • 属性

    • 房间号
    • player number
    • number of each role
    • time //记录白天黑夜
    • day //记录第几天
    • death //角色ID
      • good 好人人数
      • evil 坏人人数
    • alive //存活人数
      • good 好人人数
      • evil 坏人人数
    • number of vote

  • 方法

    • kill vote // 杀死投票后得票数最高的人（村民，狼人投票杀人）

    • change daytime //昼夜交替

    • change day //递增天数

    • result //比较alive 好人和坏人人数，death好人和坏人人数，依据规则判定输赢

      白天和晚上的流程分别作为方法，做成day(),night()

创建者

  • 创建房间，获得房间号
  • 设置房间信息
  • 推进游戏（提交入夜状态）

加入者

  • 获得房间号
  • 获得身份
  • 获得房间信息（身份配置，白天黑夜）

推进游戏

创建者开始游戏，进入第零夜。
依据流程图开始流程。

夜晚

  1. 特殊神发动技能
    • 守卫修改守护人id
    • 丘比特 恋人id
    • 野孩子 榜样id
    • 等等
  2. 狼人杀人
    • 发起投票
    • 记录投票数
    • 比较投票数，投票数高死亡
    • 计时，超时未投票、投票分散，视为空刀
    • 修改被杀玩家状态
  3. 神发动技能
    • 发动技能
    • 获取相应信息或改变角色状态
  4. 夜晚结束

白天

  1. 昨夜情况
    • 死者信息
    • 分频浪静
  2. 发言
  3. 投票
    • 记录投票
    • 比较得票数，产生死者
    • 警长记1.5票
  4. 死者遗言
  5. 白天结束

以上是大致流程，开始制作后补充细节。
